Annotations -
@Getter - Adds getter methods
@Setter - Adds setter methods
@NoArgsConstructor - creates no arguments constructor
@AllArgsConstructor - creates ALL arguments constructor
@ToString 
@EqualsAndHashCode - Include @equals and @hashcode
@AllArgsConstructor, @RequiredArgsConstructor and @NoArgsConstructor -- Included
@Log -- Added 
@Slf4j -- Requires dependency logback-classic
@Log4j -- Requires dependency apache log4j --> and basic configurator logger -- Included
@Data - Added
@Builder - Added
@Value -- For immutable class - Not included



About JSON -
Its data exchange light weight format
Easy of read/write
Not language dependent
JSON object, array, number, string, boolean (true/false) or null.



Lombok installation -
